Analyze and improve your content's readability with real-time metrics and visual feedback.

== Description ==

WordGrade provides real-time readability analysis for your WordPress content. As you write in the block editor, WordGrade calculates multiple readability metrics and presents them in a visually appealing interface to help you create more accessible content.

= Features =

* Real-time readability analysis as you write
* Visual representation of readability scores with color-coded meters
* Multiple readability metrics:
  * Flesch Reading Ease
  * Flesch-Kincaid Grade Level
  * SMOG Index
  * Coleman-Liau Index
  * Automated Readability Index
  * Linsear Write Formula
  * RIX Index
* Estimated reading time and word count
* Collapsible advanced metrics section
* Full integration with the WordPress Block Editor

= Why Readability Matters =

Good readability improves user engagement and makes your content accessible to a wider audience. WordGrade helps you identify and fix issues that might make your content difficult to read, such as:

* Sentences that are too long
* Complex words with too many syllables
* Dense paragraphs with high reading level requirements

By making your content more readable, you can:

* Increase user engagement and time on page
* Improve comprehension and retention
* Make your content accessible to more readers
* Potentially improve SEO outcomes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wordgrade` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the plugin by opening the block editor and looking for the WordGrade panel in the sidebar

== Frequently Asked Questions ==

= Will this work with the Classic Editor? =

No, WordGrade is designed to work specifically with the Block Editor (Gutenberg).

= How accurate are the readability metrics? =

WordGrade uses standard formulas for calculating readability. These metrics provide a good approximation of how readable your content is, but they are not perfect. Use them as guidelines rather than absolute measures.

= Does this plugin slow down the editor? =

WordGrade uses debounced calculations and efficient algorithms to ensure minimal impact on editor performance. You shouldn't notice any significant slowdown while using it.
